## 🏗️ 技术决策记录

### 1. 目录结构
- `core/agenthub/` 作为 Python 包根目录
- `core/agenthub/core/` 下分 agent/memory/skill/database 子模块
- `dev/` 放开发文档和交接信息（不提交 Git）

### 2. Skill 规范 v2.0
- 采用 YAML frontmatter 格式
- triggers 用复数（对齐社区标准）
- metadata.version 代替独立的 version 字段
- 保留 xavier 扩展字段（children/parent）

### 3. 适配器模式
- 每个 AI 工具对应一个 Adapter
- BaseAgentAdapter 定义统一接口
- Router 根据任务特征选择合适 Adapter
