## 二、Agent 框架目录结构

### OpenClaw (~/.openclaw)

```
~/.openclaw/
├── agents/              # Agent 配置
├── bin/                 # 可执行文件
├── browser/             # 浏览器集成
├── canvas/              # 画布
├── completions/         # 补全缓存
├── config-backups/      # 配置备份
├── credentials/         # 凭证
├── cron/                # 定时任务
├── delivery-queue/      # 消息队列
├── devices/             # 设备管理
├── extensions/          # 扩展
├── feishu/              # 飞书集成
├── identity/            # 身份认证
├── logs/                # 日志
├── media/               # 媒体文件
├── memory/              # 记忆系统 (SQLite)
├── openclaw-weixin/     # 微信集成
├── qqbot/               # QQ 机器人
├── shared-skills/       # 共享 Skills
├── skills/              # 用户 Skills
├── subagents/           # 子 Agent
├── tasks/               # 任务管理
└── workspace/           # 工作区
```

### WorkBuddy (~/.workbuddy)

```
~/.workbuddy/
├── binaries/            # 二进制文件
├── connectors/          # 连接器
├── connectors-marketplace/
├── extensions/          # 扩展
├── inspiration/         # 灵感库
├── plugins/             # 插件
├── skills/              # Skills
└── skills-marketplace/  # Skills 市场
```

### Copaw (~/.copaw)

```
~/.copaw/
├── active_skills/       # 活跃 Skills
├── bin/                 # 可执行文件
├── embedding_cache/     # 向量缓存
├── file_store/          # 文件存储
├── media/               # 媒体
├── memory/               # 记忆系统
├── sessions/            # 会话
├── tool_result/         # 工具结果
└── venv/                # Python 虚拟环境
```

### Xavier (~/.xavier)

```
~/.xavier/
├── agents/              # Agent 配置
├── memory/              # 记忆系统
│   ├── short-term/      # 短期记忆 (L0/L1/L2)
│   ├── long-term/       # 长期记忆
│   ├── knowledge/       # 知识库
│   └── _index/          # 索引
├── skills/              # Skills
├── .obsidian/           # Obsidian 配置
├── 画像/
├── 密钥/
├── 文档/
└── 项目/
```

### Qoder (~/.qoder)

```
~/.qoder/
```
