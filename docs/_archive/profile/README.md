# 用户画像模板

> ⚠️ **模板目录**: 本目录包含用户画像模板，不包含任何私人信息。
> 请复制此模板到你的 `~/.agenthub/profile/` 并填写你的真实信息。

## 目录结构

```
profile/
├── identity.yaml     # 身份信息模板
├── skills.md        # 技能图谱模板
├── projects.md      # 项目经验模板
├── schedule.md      # 课程表/日程模板
├── contacts/        # 联系人模板
├── health/          # 健康记录模板
├── growth/          # 成长规划模板
└── learning/        # 学习资料模板
```

## 使用方法

1. 复制模板到你的配置目录：
   ```bash
   cp -r ~/.agenthub/profile ~/profile-backup
   ```

2. 编辑 `identity.yaml` 填入你的真实信息

3. 根据需要编辑其他模板文件

## 隐私说明

- 所有模板文件中的内容均为**占位符示例**
- 实际使用时需要替换为你的真实信息
- 敏感信息（如密码、密钥）不要存放在此目录
