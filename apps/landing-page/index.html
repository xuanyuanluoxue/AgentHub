AgentHub 致力于构建跨平台统一规范，实现 Skill · Agent · 用户画像 · 记忆系统 四大共享生态。 一次编写，到处运行。

快速开始 View on GitHub 
$ agenthub clawhub search github
✅ Found 10 skills

$ agenthub clawhub install github
✅ github v1.0.0 installed

$ agenthub skill list
📦 agent        v1.0.0
📦 hello-world   v1.0.0
📦 github        v1.0.0

四大核心卖点

一次编写，到处运行

共享 Skill 库

统一格式，跨平台兼容。YAML frontmatter + Markdown，工具无关。

✓ OpenClaw / OpenCode / ClawHub / Hermes 通用
✓ SemVer 版本控制
✓ 无需 API Key，直接安装 ClawHub 商店 Skills

共享 Agent 库

专业角色，智能路由。Router + Specialist 双类型定义。

✓ 自动加载所需 Skills
✓ 短期/长期记忆策略可定制
✓ 跨工具共享

共享用户画像

工具无关，永久携带。YAML + Markdown 纯文本格式。

✓ 身份信息、联系方式、社交账号
✓ 审美偏好、沟通风格、回复习惯
✓ 常用路径、项目目录、知识库

共享记忆系统

持久记忆，持续学习。多层架构 + 向量检索。

✓ 短期 / 长期 / 知识库 三层
✓ 基于语义相似度的记忆召回
✓ 跨工具同步

统一目录结构

简洁清晰，职责分明

~/.agenthub/
├── core/              # Python 包
│   └── agenthub/          # CLI + 核心模块
├── skills/             # 已安装 Skills
├── agents/             # Agent 配置
├── profile/            # 用户画像
├── apps/               # 独立应用
│   ├── dashboard/        # Web 管理界面
│   └── tool-detector/    # AI 工具检测
└── docs/               # 规范文档

快速开始

一行命令即可安装

🪟

Windows

irm https://raw.githubusercontent.com/xuanyuanluoxue/AgentHub/main/scripts/install.ps1 | iex
🐧

Linux / macOS / WSL

curl -fsSL https://raw.githubusercontent.com/xuanyuanluoxue/AgentHub/main/scripts/install.sh | bash
☁️

安装 ClawHub 商店 Skills

无需 API Key 
agenthub clawhub search github    # 搜索
agenthub clawhub install github   # 安装
