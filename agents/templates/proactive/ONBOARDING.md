# ONBOARDING.md - Agent 初始化向导

> 版本: v1.0 | 更新: 2026-04-27
> **来源**: proactive-agent (Hal Labs)
> **用途**: AgentHub Agent 初始化向导模板

## 如何使用

此文件跟踪初始化进度。不要删除它——Agent 使用它来恢复。

---

## 状态

- **状态**: not_started
- **进度**: 0/12 核心问题
- **模式**: interactive（或者：drip）
- **最后更新**: —

---

## 如何工作

当 Agent 看到状态为 `not_started` 或 `in_progress` 的此文件时，它知道要帮助完成设置。用户可以：

1. **交互模式** — 在一个会话中回答问题（约 10 分钟）
2. **滴灌模式** — Agent 在几天内自然地提问 1-2 个问题
3. **暂时跳过** — Agent 立即工作，从对话中学习

说"开始初始化"来开始，或"稍后问我"来滴灌。

---

## 核心问题

回答这些问题来帮助 Agent 了解用户。留空可以跳过。

### 1. 身份
**我该怎么称呼你？**
>
**你的时区是什么？**
>

### 2. 沟通
**你希望我怎么沟通？（直接/详细/简洁/随意）**
>
**有什么我应该避免的禁忌吗？**
>

### 3. 目标
**你现在的首要目标是什么？（1-3句话）**
>
**1年后"成功"是什么样的？**
>
**当你成功时，理想的生活是什么样的/感觉如何？**
>

### 4. 工作风格
**你什么时候最高效？（早上/下午/晚上）**
>
**你更喜欢异步沟通还是实时沟通？**
>

### 5. 上下文
**你目前正在做什么？（项目、工作等）**
>
**我应该了解的你工作/生活中的关键人物是谁？**
>

### 6. Agent 偏好
**你的 Agent 应该有什么样的性格？**
>

## 初始化之后

一旦完成（或收集到足够的答案），Agent 将：
1. 使用用户的上下文更新 USER.md
2. 使用个性偏好更新 SOUL.md
3. 将状态设置为 `complete`
4. 开始主动模式

用户可以通过编辑此文件或告诉 Agent 来随时更新答案。
