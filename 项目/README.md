# 项目配置

## 🎯 项目愿景

**AgentHub** → 图形化 AI 工具管理平台

> 一个中央化的 AI 工具管理与部署系统，支持一键部署、网关接入、多端交互

### 核心定位

| 定位 | 说明 |
|------|------|
| **工具管理中心** | 统一管理 OpenCode、OpenClaw、CodeBuddy 等 AI 工具的配置 |
| **一键部署** | 支持 AI 工具的快速部署和初始化 |
| **网关接入** | 接入微信、QQ、Telegram 等消息网关 |
| **多端交互** | 支持 Web、TUI、CLI、GUI 多种交互方式 |
| **资源中心** | 中央化 Skill 库、Agent 库、提示词库管理 |
