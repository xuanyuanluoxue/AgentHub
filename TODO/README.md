# TODO 使用说明

> 版本: v2.0 | 更新: 2026-05-15

---

## 快速开始

### 创建待办

在 `TODO/待办/` 目录创建文件，文件名格式：

```
{序号}-{标题}.md
```

### 开始任务

将文件从 `待办/` 移动到 `进行中/`，更新 frontmatter 中的 `status: "in_progress"`。

### 完成任务

1. 将文件移动到 `TODO/归档/` 目录
2. 更新 frontmatter 中的 `status: "completed"`

---

## 文件模板

```markdown
---
id: "001"
title: "任务标题"
status: "pending"
priority: "high"
created_by: "agent-id"
created_at: "2026-04-29T10:00:00Z"
updated_at: "2026-04-29T10:00:00Z"
---

# 001 - 任务标题

## 任务描述

描述任务内容。

## 子任务

- [ ] 子任务1
- [ ] 子任务2

## 备注

补充信息。
```

---

## 优先级说明

| 优先级 | 使用场景 |
|--------|----------|
| `high` | 用户明确要求、有截止日期、阻塞后续 |
| `medium` | 重要但不紧急 |
| `low` | 可选功能 |

---

## 状态说明

| 状态 | 目录 | 说明 |
|------|------|------|
| `pending` | `待办/` | 待开始 |
| `in_progress` | `进行中/` | 进行中 |
| `homework` | `作业/` | 作业任务 |
| `completed` | `归档/` | 已完成 |
| `cancelled` | `归档/` | 已取消 |
